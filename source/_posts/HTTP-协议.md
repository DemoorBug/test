# HTTP CODE
100-199之间代表操作要持续进行，接下来要做一些其他事情才会返回给你
200-299 代表操作成功
300-399 代表操作重定向，有别的方式获取数据
400-499 发送请求有问题，401发送请求的时候没有认证，没有权限获取资源的内容
500-599 服务器出现问题
